# Employee Profile Management System

This is a Python-based Employee Profile Management System developed as an academic/internship project.

## Features
- Add new employee profile
- View all employee profiles
- Search employee by ID
- Update employee details
- Delete employee profile

## Technologies Used
- Python 3
- File Handling
- Modular Programming
- Exception Handling

## How to Run
1. Install Python 3
2. Download or clone the repository
3. Run the file: EmployeeMain.py
4. Follow the on-screen menu
